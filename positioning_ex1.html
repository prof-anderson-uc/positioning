Relative & Absolute Positioning

The position property determines how an element is positioned on a page and whether or not it will appear in the normal flow, be offset a little bit, or free to be placed anywhere. Absolute and Relative positioning works with the box offset properties - top, right, bottom, left - which identify where the element will be either positioned or moved from.

By default every element is position: static which means that it exists in the normal flow of the page and does accept any of the box offset properties. Top, right, bottom, and left don't work on "normal" (static) elements, only ones that are relative or absolute.
